# Karliq

Statisk one-page hemsida för Karliq.

## Öppna lokalt

Öppna `index.html` direkt i webbläsaren.

## Innehall

- `index.html` - struktur och text
- `styles.css` - design, responsiv layout och animationer
- `script.js` - reveal-animationer och interaktiva glow-effekter
- `karliq-logo.svg` - Karliq-logotyp
- `robots.txt` - instruktioner för sökmotorer
- `sitemap.xml` - sitemap för `https://karliq.me/`

## Deployment senare

Sidan är förberedd för `https://karliq.me/` och kan läggas på GitHub Pages, Netlify, Vercel eller annan statisk hosting.
